# AGENTS.md

## 协作规则

- 在本仓库内执行任务时，除本文件外，也默认遵守根目录的 `CLAUDE.md`。
- 如果 `AGENTS.md` 与 `CLAUDE.md` 存在重叠规则，优先执行要求更严格、信息更完整的那一条。
- 如果仓库规则与更高优先级的系统约束冲突，需要先向用户明确说明冲突点，再按更高优先级规则执行。

## 提交规范

- 所有新的 Git commit 信息必须使用中文。
- commit 标题需要直接说明本次改动的主题，避免使用空泛描述。
- commit 正文至少说明三件事：问题是什么、具体如何修复、最终影响或结果是什么。
- 如果一次提交覆盖多个问题，正文中需要分别写清，不要用“顺手调整”“若干优化”这类模糊表述。
